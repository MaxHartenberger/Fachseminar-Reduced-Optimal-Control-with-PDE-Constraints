Exercise Sheet 3
Submission Deadline: May 28th, 00:00h
On this sheet we will solve a linear-quadratic elliptic optimal control problem by the so-called control reduced
approach. In this formulation, the PDE constraint is eliminated by the solution operator S, which leads to an
unconstrained optimiaztion problem in the control space U(see below). This is different to so-called all-at-once
approach, which we discussed on the last sheet in a discretize-then-optimize setting. There the PDE constraint
was not eliminated, but the whole optimality system (including Lagrange multipliers) was directly solved.

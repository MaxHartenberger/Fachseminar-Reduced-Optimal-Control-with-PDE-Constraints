# Beamer presentation

This folder contains a Beamer slide deck that reuses figures and tables already generated by the project.

## Build

From the repo root:

```bash
cd presentation
pdflatex -interaction=nonstopmode -halt-on-error presentation.tex
pdflatex -interaction=nonstopmode -halt-on-error presentation.tex
```

This produces `presentation.pdf`.

## Notes

- Figures are included via relative paths from `../plots` and `../mesh`.
- Tables are included via `\input{../results/mesh_independence_table.tex}` and `\input{../results/mesh_iterations_table.tex}`.
